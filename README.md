# 扎扎师幸存者
#### 介绍
南开大学2026c++图形化编程大作业项目
基于Qt和c++语言进行开发的类幸存者游戏：扎扎师幸存者
#### 软件架构
##### 1.概述与背景
扎扎师是以二战为背景的卡牌游戏kards中的一张波兰盟国卡牌，最初因其幽默的特效而被玩家拿来制造梗，从而延伸出kards中的“宝马主义”，令人忍俊不禁。我贯彻着kards中“宁骑一日宝马，不做百年小逼”的原则，构思出这款以kards为ip，主角为扎扎师的类幸存者游戏
##### 2.约束条件
使用c++语言中的类和对象知识进行开发
##### 3.核心模块设计
1. 以kards中多国(苏联、芬兰等)卡牌为原型设计了几种武器，武器可以在局内获取和升级
2. 以日本国家卡牌为原型设计了不同的敌人类型，敌人数量随时间增加，生成速度对时间加快
#### 安装教程
1. 点击右侧发行版中蓝色字体，找到对应版本即可下载
#### 使用说明
下载解压完成后找到.exe文件点击即玩
